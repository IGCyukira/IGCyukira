静寂の中から、それは生まれた。

空を舞う雪は紙のように薄く、音もなくこの半透明の灰色の遺構へと降り積もっていく。

その遺構の中心に、一粒の人影が横たわっていた。

目に見えぬ刃のような寒風が肌を削らなければ、彼女は自分がこの空間に存在していることさえ確信できなかっただろう。完璧な幾何学構造を持つ雪の結晶たちが彼女の傍らで舞い、積み重なり、まるで沈黙の衛兵のように、未だ目覚めぬ**君主（くんしゅ）**を護っていた。

しかし、その君主は一向に瞳を開こうとはしない。彼女は依然として静かに横たわり、夢を見ることもなく、息を潜めたまま。彼女はこの混沌とした世界に忘れ去られた注釈のようでもあり、あるいは世界すべてが息を呑み、永きにわたって待ち続けた「答え」のようでもあった。

雪は降り続く。それはまるで、透き通った永遠の夢のように。

眠れる暗闇の中で、彼女は幾星霜もの時を超えた未来の投影を見ていた。

それは行き交う人々の群れ、共に歩む仲間たち、溢れるような笑い声……。遠くへと続く迷い子の森の小径、霧の中に見え隠れする山々の稜線。茶屋の喧騒と公園の静寂が、一枚の絵画のように織りなされていた。

それら美しい欠片は流光のように瞬き、あまりに速く、あまりに急で、彼女が捉える術（すべ）はなかった。

――その時、情景は突如として反転する。

彼女の目に映ったのは、崩れゆく天空、墜ちゆく星々。名もなき何らかの異常の中で、世界すべてが虚無へと帰していく光景だった。

遺構に響き渡る清らかな鐘の音が、永い夢を打ち砕いた。

彼女はゆっくりと瞳を開いた。その空色（そらいろ）の瞳には、夢の余温がまだ残っている。彼女はただ、目の前に広がるこの純粋で、美しい世界を、じっと見つめていた。

## Another IGCyukira: [🍬 IGCyukito](./README-IGCyukito.md)

</div>

<br>

IGCyukito は、決して見知らぬ存在ではありません。
彼女は、陽だまりの中で無邪気に笑う Yukira そのもの。
ただ、あなたの視線が本当に向けられた瞬間、
ふっと頬を赤らめて、言いかけた言葉を胸の奥にしまい込み、
隠しきれない小さな照れを覗かせるだけなのです。

元気いっぱいの Yukira を、きっとあなたも知っているでしょう。
人混みの中で手を引いて走り、
風鈴のように澄んだ声を響かせるあの子。
けれど、ふと振り返ったときに見えるのが Yukito です。
賑わいが去ったあと、
そっとあなたの服の裾をつまみ、背中に隠れる小さな影。
見知らぬ人と目を合わせるのは苦手でも、
あなたにだけは、すべての信頼を預ける臆病な子。

甘えたい気持ちはあるのに、言葉にできなくて。
膝を抱え、顔をマフラーに埋めながら、
潤んだ大きな瞳で、こっそり、こっそりとあなたを見る。
まるで溶けかけの小さなキャンディのように。

彼女には、世界を抱きしめるほどの勇気はありません。
だからこそ、その勇気を少しずつ大切に集めていきます。
静かな場所で、
あなただけに届く、やわらかく小さな声で——
そっと、あなたの名前を呼ぶために。
